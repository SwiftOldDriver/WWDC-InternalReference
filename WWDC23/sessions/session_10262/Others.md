## 不超过 120 个字的文章简介

本文主要是介绍 Safari 调试工具集，以及 WWDC23 中新增的一些调试功能，为开发者提供更全面、完善的调试流程。第一部分是介绍 Web Insepector 常规使用，以及简要介绍了下本次新增的调试工具；第二部分是介绍了 Safari 的响应式设计模式的使用，以及本次开始支持在 Safari 中调起模拟器调试 Web 内容，实现无真机测试；第三部分是介绍了 xrOS 新系统在 Mac 中如何配对调试，以及 Web 新增的 API 使用说明；第四部分是演示了如何使用 WebDriver 实现自动化测试；第五部分是对 Web 未来的一些探索，介绍了新增的 Feature Flags 及其状态，方便开发者可以提前了解 Web 的未来发展方向。
