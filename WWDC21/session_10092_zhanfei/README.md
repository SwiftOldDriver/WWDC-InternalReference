### 标题

了解并使用 Wallet 和 Apple Pay

### 简介

本文通过回顾 WWDC 2021 了解 Wallet 和 Apple Pay 增加和更新的新功能。分享 Wallet 和 Apple Pay 的应用场景以及展示对应的支持范围。最后通过支付场景实现还原从支付环境配置、Xcode 项目配置、代码实现的整个过程。并提供相关 Demo 以供大家测试。
