## 文章简介

WWDC21 10141

探索如何利用 HLS 播放列表标签和携带您内容的调度清单，来针对每一位浏览者动态地更新 CDN 冗余策略。了解如何进行负载均衡、故障转移等。

WWDC21 10142

探索如何创建在剧集、歌曲、场景之间无缝转换的流媒体内容。 通过无缝 HLS 播放，您可以将多条内容动态地拼接在一起。您可以用来定制您的锻炼、设计交互式内容、讲述有趣的故事等等。 

我们将向您展示如何为流媒体音乐、事件的记录和预先录好的视频提供实实在在的连续性，并在您的 App 中提供引人入胜的观看体验。

WWDC21 10143

了解如何使用 AVFoundation API 来突出 App 内容的不同变体。 我们将向您展示如何使用这些 API 来检查 HLS 内容的不同视频特性，包括 SDR/HDR、FPS 等属性。 我们将探索代表流媒体和离线内容的 AVAssetVariant。
