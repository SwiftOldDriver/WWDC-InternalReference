## 简介

Swift 5.5 创新性地引入了结构化并发的概念。同时，也革命性地引入了不同风格的任务、任务树的自动管理机制以及编译时的数据安全检查，极大地降低了并发程序编写难度。尽管最终正式版还是存在更改部分 API 的可能，我们仍然推荐大家先学习并熟悉一下编写 Swift 程序的新方法。
