### 标题

WWDC21 10076 - 使用 Object Capture 创建 3D 模型

### 简介

Apple 自 2017 年推出 ARKit 以来一直在推动增强现实技术在其各个平台上的应用和发展，但 AR 内容的匮乏与高成本一直是制约开发者们投入 AR 的重要因素。在本次 WWDC 上，Apple 将 3D 模型的构建效率提升了到了前所未有的高度，RealityKit 2 框架中的新 Object Capture API 为我们提供了一种非常便捷的方式来为生活中的物体创建 3D 模型，而你需要的仅仅是为目标物拍摄几张照片。此外，配合 AR Quick Look 可以轻松将创建好的模型集成到你的应用之中。
