## 标题

WWDC21 10253 - 使用 Swift 特性 Result Builder 定制 DSL

## 文章简介

DSL 全称是 **Domain-Specific Language**，中文翻译为**领域特定语言**。目前我们可以使用 Swift 语言作为宿主语言的语法特性来编写 DSL 的语法规则，通过使用 Result Builder、Modifier-style Method、Property Wrapper 等特性可以编写出十分精简且可读性极强的 DSL。

通过阅读此文，你可以学习到如何使用 Swift 定制适于自身业务的 DSL，并且可以了解到 SwiftUI 的 DSL 原理以及深入 Result Builder 特性。
