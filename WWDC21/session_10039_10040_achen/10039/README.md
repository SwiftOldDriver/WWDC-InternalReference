### 标题
Create ML新妙用：手势分类和手部动作分类

### 简介
2018年WWDC苹果推出了Create ML框架；它极大的降低了机器学习的入门门槛，通过该框架，一个没有机器学习基础的苹果开发者也可以快速训练出自己的机器学习模型并将其应用到iOS/Mac应用中，并且训练出来的模型在模型大小和运行速度方面都具备一定的优势。过去几年，Create ML和相关框架也得到了长足的发展，其能力逐步细化到了各个实际应用场景中。今年，苹果给我们带来了新的应用场景：区分不同的手势和手部动作，同时还支持左右手的识别。在这篇文章中，我们一起来看看今年苹果带给我们的惊喜吧。
