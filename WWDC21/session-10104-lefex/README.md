### 标题

WWDC21 - 在 iOS 中给 Safari 开发插件

### 简介

浏览器插件如同一把利剑能够插入到网页中，可以对网页进行一些操作，比如给某个网站添加菜单，写一个抢票插件，总之一些自动化的操作都可以通过插件来实现。以前只有 mac 和 ipad 支持浏览器插件，现在 iOS15 也支持了。
